# Installation
```
1. git clone https://github.com/wahyunanangwidodo/nvim.git "$env:LOCALAPPDATA\nvim"
2. git clone https://github.com/wbthomason/packer.nvim "$env:LOCALAPPDATA\nvim-data\site\pack\packer\start\packer.nvim"
3. :PackerInstall
4. :PackerCompile
5. Uncomment "vim.cmd [[colorscheme gruvbox-material]]" //settings.lua
```
